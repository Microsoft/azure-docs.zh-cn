# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Azure 安全基准 V2 和 CIS Microsoft Azure 基础基准的映射

CIS Microsoft Azure 基础基准是 Internet 安全中心提供的安全指导原则，旨在建立 Azure 的安全基线配置。 该基准的范围是在采用 Azure 云的同时建立基本的安全级别。 此基准与 [Azure 安全基准 v2](overview.md) 相一致。

我们提供了 [CIS Microsoft Azure 基础基准最新版 v1.3.0](https://www.cisecurity.org/benchmark/azure/) 与 [Azure 安全基准 v2](overview.md) 之间的映射。 这有助于你了解有关这两套基准的建议之间的相符程度。 可在[此处](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)下载完整映射。
