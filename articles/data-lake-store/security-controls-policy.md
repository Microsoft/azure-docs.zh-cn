# <a name="azure-policy-regulatory-compliance-controls-for-azure-data-lake-storage-gen1"></a>Azure Data Lake Storage Gen1 的 Azure Policy 法规遵从性控制措施

[Azure Policy 中的法规遵从性](../governance/policy/concepts/regulatory-compliance.md)为与不同合规性标准相关的“合规性域”和“安全控制”提供由 Microsoft 创建和管理的计划定义，称为“内置项” 。 此页列出了 Azure Data Lake Storage Gen1 的“合规域”和“安全控制措施” 。
可以分别为“安全控件”分配内置项，以帮助 Azure 资源符合特定的标准。
