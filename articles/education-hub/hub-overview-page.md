# <a name="managing-your-academic-grant"></a>管理学术授权

Azure 教育中心的主要登录页是概述页。 此页包含学术授权的所有相关信息，例如已建立的教室的数量，以及从这些教室分配并使用的总运行额度。 此页还显示教育中心提供的其他权益的快捷方式：软件、学习和快速启动模板。

## <a name="overview-page"></a>概述页
:::image type="content" source="media/hub-overview-page/overview.png" alt-text="登录错误消息。" border="false":::

1. “课程”显示讲师帐户的活动课程总数。
1. “实验室”显示已分配给学生的活动实验室的总数。
1. “所需操作”列出了需要完成的任何操作，例如接受实验室邀请。
1. “软件”列出了教师可下载的免费软件。
1. “学习”通过 Microsoft Learn 链接到免费的 Azure 学习路径。
1. “快速启动模板”包括 Azure 模板，可帮助加快和简化常见任务的部署。
