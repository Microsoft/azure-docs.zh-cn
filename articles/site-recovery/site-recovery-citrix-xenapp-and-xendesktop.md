# <a name="end-of-support-for-disaster-recovery-of-citrix-workloads"></a>终止对 Citrix 工作负载灾难恢复的支持

从 2020 年 3 月起，Citrix 已宣布弃用并终止对公有云托管工作负载的支持。 因此，不建议使用 Site Recovery 来保护 Citrix 工作负载。
