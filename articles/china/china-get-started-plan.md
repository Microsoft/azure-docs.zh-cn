# <a name="plan-an-azure-deployment"></a>规划 Azure 部署
从创立中国商店到在世纪互联运营的 Microsoft Azure（Azure 中国世纪互联）中托管工作负荷，组织需要做出许多决策：
- 提前规划 [ICP 备案和注册](/azure/china/china-overview-policies)流程。
- 检查 Azure 运营商[世纪互联](/azure/china/china-overview-operations)提供的服务级别协议 (SLA)。
- 确保 Azure 应用程序或工作负荷[符合](/azure/china/china-overview-policies)国家、地区和行业特定的要求。
- 了解中国的[合作伙伴](/azure/china/china-resources-partners)和 Azure 市场产品/服务。
- 在 Azure 中[开发](/azure/china/china-get-started-developer-guide)、[测试](/azure/china/china-how-to-manage-performance)和[托管](/azure/china/china-overview-service-hosting)应用程序。
