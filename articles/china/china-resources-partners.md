# <a name="azure-china-21vianet-partners"></a>Azure 中国世纪互联合作伙伴
本页根据具体的需求，提供世纪互联运营的 Microsoft Azure（Azure 中国世纪互联）合作伙伴的各个联系点。

### <a name="local-china-partner-solutions"></a>当地的中国合作伙伴解决方案
- 若要通过认证的合作伙伴购买间接联机服务高级协议 (OSPA)，请与当地的 Microsoft 代表联系。
- 找到一家 [Azure 认证合作伙伴](https://www.azure.cn/partnerancasestudy/partner/)，帮助自己购买符合要求的本地化解决方案。
- 从 [Azure 中国市场](https://market.azure.cn/)获取随时可供部署的解决方案，以及经过优化的集成式应用程序和服务。
- 可考虑从 [Azure 市场](https://azure.microsoft.com/marketplace/)获取全球合作伙伴解决方案，但是，由于大多数解决方案数据尚未专门针对 Azure 中国世纪互联进行配置，因此，该解决方案预期只能解决迁移问题。
- 使用[内容分发网络](/azure/china/china-get-started-service-cdn) (CDN) 合作伙伴或服务来优化对中国境内的网站流内容的访问。
