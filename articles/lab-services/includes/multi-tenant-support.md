如果你在两个或更多 Azure AD 租户中具有管理员或实验室所有者角色，则可以通过选择右上角的控件在 Azure 实验室服务门户中的租户之间进行切换，如以下屏幕截图所示： 

![用于在 Azure 实验室服务门户中的租户之间进行切换的控件的屏幕截图。](../media/multi-tenant-support/picker.png)

选择 Azure AD 租户后，可选择左上方的控件来切换到实验室帐户。
