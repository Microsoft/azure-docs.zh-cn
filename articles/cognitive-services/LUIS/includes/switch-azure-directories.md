如果没有看到你创建的应用或与你共享的应用，则可能需要切换到其他 Azure 目录。

1. 单击屏幕右上角的头像。 然后单击“切换到其他 Azure 目录”。 
2. 在出现的窗口中，确保选择包含与你共享的 LUIS 资源的 Azure 目录。
