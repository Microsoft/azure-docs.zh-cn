# <a name="assign-user-roles-and-permissions"></a>为用户分配角色和权限

**相应的角色**

- 所有者
- Manager

为了访问与市场或开发人员帐户相关的功能，你需要为自己分配以下一个或多个角色。 每个角色都有一组特定的权限，可以在帐户中执行某些功能。

| 角色 | 说明 |
| ------------ | ------------- |
| 所有者 | 创建发布者帐户时，会将所有权角色分配给合作伙伴。<br><br>权限详细信息：<br><br>&#10004;&#160;&#160;帐户的完整所有权。<br>&#10004;&#160;&#160;添加用户类型<br>&#10004;&#160;&#160;可以完成付款和税务信息<br>&#10004;&#160;&#160;查看反馈报告和摘要<br>&#10004;&#160;&#160;将发布者帐户注册到任何其他开发人员计划中 |
| Manager | &#10004;&#160;&#160;管理发布者帐户用户、角色和权限<br>&#10004;&#160;&#160;管理租户<br>&#x2718;&#160;&#160;访问税务和付款设置 |
| 开发人员 | &#10004;&#160;&#160;上传包<br>&#10004;&#160;&#160;提交应用和附加产品<br>&#10004;&#160;&#160;查看遥测详细信息的使用情况报告<br>&#x2718;&#160;&#160;访问财务信息或帐户设置 |
| 业务参与者 | &#10004;&#160;&#160;访问财务信息<br>&#10004;&#160;&#160;设置定价详细信息<br>&#x2718;&#160;&#160;创建或提交新的应用和附加产品 |
| 财务参与者 | &#10004;&#160;&#160;查看付款报告<br>&#10004;&#160;&#160;管理付款和税务配置文件<br>&#x2718;&#160;&#160;对应用或设置进行更改 |
| 营销人员 | &#10004;&#160;&#160;回复客户评论<br>&#10004;&#160;&#160;查看非财务报告<br>&#x2718;&#160;&#160;对应用或设置进行更改 |
|||

> [!NOTE]
> 对于商业市场计划，不使用全局管理员、业务参与者、财务参与者和营销人员角色。 将这些角色分配给用户不起作用。 只有管理员和开发人员角色向用户授予权限。

有关在合作伙伴中心的其他区域（例如 Azure Active Directory (AD)、云解决方案提供商 (CSP)、控制面板供应商 (CPV),、来宾用户或 Microsoft 合作伙伴网络 (MPN)）中管理角色和权限的详细信息，请参阅[在合作伙伴中心分配用户角色和权限](/partner-center/permissions-overview)。

> [!NOTE]
> 对这些行执行的任何用户管理、角色分配活动都将位于你所在的帐户的上下文中。 如果需要管理不同的帐户，请参阅有关在卖方之间切换的部分。
