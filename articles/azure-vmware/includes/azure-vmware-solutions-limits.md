下表描述了 Azure VMware 解决方案的最大限制。

| **资源** | **限制** |
| --- | --- |
| 每个私有云的群集 | 4 |
| 每个群集的最小节点数 | 3 |
| 每个群集的最大节点数 | 16 |
| 每个私有云的节点数 | 64 |
|  每个私有云的 vCenter | 1  |
| HCX 站点对 | 3 with Advanced edition，10企业版 |
| ExpressRoute 限制 | 4 SDDCs，10 SDDCs |
| 通过 vWAN 公开的公共 Ip | 256 |
|  vSAN 容量限制 | 75% 的可用 (保留适用于 SLA 的 25%)   |
