# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>如何在 Azure Data Share 中删除向收件人发送的邀请

本文介绍如何在收件人接受邀请之前删除向收件人发送的邀请。 此操作将阻止他们创建共享订阅，他们将接收不到任何共享的数据。 如果删除向已创建共享订阅的使用者发送的邀请，则删除其邀请不会产生任何影响。

## <a name="navigate-to-a-sent-data-share"></a>导航到已发送的数据共享

在 Azure Data Share 中，导航到已发送的共享并选择“邀请”选项卡。此处会列出已针对此数据共享向其发送邀请的所有收件人。 选中要删除其邀请的收件人旁边的复选框，然后单击“删除”。
