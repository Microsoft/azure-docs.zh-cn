# <a name="texconv---texture-conversion-tool"></a>TexConv - 纹理转换工具

TexConv 工具是一个外部命令行工具，用于执行批量纹理转换。
可在此 git 存储库中找到该工具：[用于 TexConv 工具的 GitHub 存储库](https://github.com/microsoft/DirectXTex/wiki/Texconv)。

对于远程渲染，不必转换纹理，因为转换步骤可以处理常见的图像文件格式，并将纹理转换为适当的运行时格式。 但是，此工具对于使用自定义设置在本地转换纹理很有帮助。
