# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>在 Azure 门户中查看实例库存

上传[指标、日志](upload-metrics-and-logs-to-azure-monitor.md)或[使用情况](view-billing-data-in-azure.md)后，可以从 Azure 门户作为 Azure 资源来查看实例。 如果要在 Azure 门户中查看资源，请转到特殊 URL [https://portal.azure.com](https://portal.azure.com) 并完成以下步骤：

1. 转到“所有服务”。
1. 搜索数据库实例类型。
1. 将类型添加到收藏夹。
1. 在左侧菜单中，选择实例类型。
1. 在与其他 Azure SQL 或超大规模 Azure PostgreSQL 资源相同的视图中查看实例（使用筛选器获得精细视图）。
