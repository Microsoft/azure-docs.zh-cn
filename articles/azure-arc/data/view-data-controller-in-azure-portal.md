# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>在 Azure 门户中查看 Azure Arc 数据控制器资源

首次完成[将指标或日志上传到 Azure](upload-metrics-and-logs-to-azure-monitor.md) 或者[上传使用情况数据](view-billing-data-in-azure.md)的操作后，便可以在 Azure 门户中查看 Azure 中创建的 Azure Arc 数据控制器资源。

使用以下专用 URL 打开 Azure 门户：[https://portal.azure.com](https://portal.azure.com)。

在搜索栏中按名称搜索你的数据控制器，并单击它。
