# <a name="find-an-administrator-to-change-my-information-in-my-account"></a>查找管理员以更改“我的帐户”中的信息

你的 IT 或人力资源部门可能会将你的一些个人资料信息保留在不同的系统中；例如，个人资料照片、姓名或职务。 如果不允许你自行更改此信息，请与你的人力资源部门或管理员联系，让他们为你更改。

## <a name="why-cant-you-just-give-me-the-name-of-who-to-contact"></a>为什么不能直接给我联系人的姓名？

我们很愿意给你联系人的姓名或电子邮件地址，但 Microsoft 没有此信息。 本文提供了一些建议，可帮助你找到你的管理员。

## <a name="how-do-i-find-out-who-my-admin-is"></a>如何找出谁是我的管理员？

当查找“我的帐户”管理员以更新某些信息、重置密码、删除帐户或执行其他任务时，请查看下面关于你应当联系的人员的一些建议：

- 大学和学校：联系技术支持团队联系。 通常，可以在大学网站上查找链接。 在较小的学校，可能只有几个拥有管理员权限的技术人员。

- 大型企业：联系组织的内部支持人员或技术支持。

- 小型企业：联系企业所有者或共同所有者。 通常，他们会为其 IT 顾问提供管理员权限，IT 顾问为其企业执行所有计算机维护工作。

如果你不知道在你的单位或学校该联系谁来获取帮助，请尝试询问为你提供“我的帐户”用户帐户和密码的人。
