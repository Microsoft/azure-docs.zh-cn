# <a name="versioning-and-support-for-azure-defender-for-iot"></a>Azure Defender for IoT 的版本控制和支持 

本文介绍了适用于 IoT 的 Defender 的支持、重大更改策略以及当前可用的用于 IoT 的 Azure Defender 版本。 

## <a name="servicing-information-and-timelines"></a>维护信息和日程表 

Microsoft 计划每季度免费发布 Azure defender for IoT 的更新。 每个公开上市 (为 IoT 传感器提供 Azure Defender) 版本，并支持在其发布后的最多9个月内的 Azure Defender 本地管理控制台。 修补程序和新功能将应用于当前提供支持的当前 GA 版本，并且不会应用于旧的 GA 版本。

## <a name="versions-and-support-dates"></a>版本和支持日期

| 版本 | 发布日期 | 结束支持日期 |
|--|--|--|
| 10.0 | 01/2021 | 10/2021 |
