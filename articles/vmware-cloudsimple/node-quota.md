# <a name="cloudsimple-node-quota-limits"></a>CloudSimple 节点配额限制

为 CloudSimple 服务启用订阅后，四个节点是可购买的默认数量。  可以从 Azure 门户购买任何[节点类型](cloudsimple-node.md)。  创建私有云至少需要三个具有相同 SKU 的节点。  如果已购买节点，则在尝试购买其他节点时可能会看到错误。

## <a name="quota-increase"></a>配额增加

可通过提交支持请求来请求增加节点配额。 服务运营团队会对请求进行评估，并与你协作来提高节点配额。  创建新票证时，请选择以下选项：

* 问题类型：技术
* 订阅：你的订阅 ID
* 服务类型：VMware Solution by CloudSimple
* 问题类型：专用节点配额
* 问题子类型：提高专用节点的配额
* 主题：提高配额

在支持票证的详细信息中，提供所需的节点数和节点 SKU。

* 节点 SKU
* 其他需要提高配额的节点的数目
