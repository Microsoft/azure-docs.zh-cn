# <a name="supported-devices"></a>支持的设备

[Azure 门户](https://portal.azure.com)是基于 Web 的控制台，可在所有新式的台式机和平板电脑设备的浏览器中运行。 若要使用门户，必须在浏览器上启用 JavaScript。

如果需要从移动设备管理 Azure 资源，请尝试 [Azure 移动应用](https://azure.microsoft.com/features/azure-portal/mobile-app/)。 它适用于 iOS 和 Android。

## <a name="recommended-browsers"></a>建议的浏览器

建议使用与操作系统兼容的最新浏览器。 支持以下浏览器：

* Microsoft Edge（最新版本）
* Internet Explorer 11 (在2021年3月31日前受支持) 
* Safari（最新版本，仅限 Mac）
* Chrome（最新版本）
* Firefox（最新版本）
