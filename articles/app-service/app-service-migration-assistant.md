# <a name="migrate-to-azure-app-service"></a>迁移到 Azure 应用服务

使用[应用服务迁移助手](https://azure.microsoft.com/services/app-service/migration-assistant/)可将本地应用迁移到 Azure 应用服务。 应用服务迁移助手旨在通过免费、简单、快速的解决方案将应用程序从本地迁移到云端，从而简化迁移到云的过程。

使用 Azure 应用服务迁移助手，可以快速执行以下操作：

- 扫描应用 URL 以评估其是否适合迁移
- 下载迁移助手以开始迁移。
- 使用该工具运行就绪检查和应用配置设置的一般评估
- 通过该工具将应用或站点迁移到 Azure 应用服务。

[观看如何将 Web 应用迁移到 Azure 应用服务](https://www.youtube.com/watch?v=9LBUmkUhmXU)。
