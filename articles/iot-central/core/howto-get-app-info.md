# <a name="about-your-application"></a>关于你的应用程序

本文介绍如何获取有关 IoT Central 应用程序的信息。 你可能需要：

- 此信息（如果你联系支持人员）。
- 你的应用程序用于在 Azure 门户中查找账单信息的 Azure 订阅。
- 应用程序的 ID（使用 REST API 时）。
- 用于完成任务（如添加连接器）的应用程序的版本。

## <a name="get-information-about-your-application"></a>获取有关应用程序的信息

若要获取有关 IoT Central 应用程序的信息，请执行以下操作： 

1. 选择顶部菜单上的“帮助”链接。

1. 选择“关于应用”。

1. “关于应用”页显示有关应用程序的信息：  

    :::image type="content" source="media/howto-get-app-info/about-your-app2.png" alt-text="“关于应用”屏幕截图":::

    使用“复制信息”按钮将信息复制到剪贴板。

现在，你已经了解了如何查找 IoT Central 应用程序的版本，建议的下一步是继续浏览面向管理员的操作方法文章：[更改 IoT Central 应用程序设置](howto-administer.md)。
