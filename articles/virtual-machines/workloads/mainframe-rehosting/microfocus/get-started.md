# <a name="micro-focus-devtest-environments-on-azure"></a>Azure 上的 Micro Focus 开发/测试环境

使用具备开发和测试环境的 Azure，是促进现代化工作的好办法。 使用合作伙伴解决方案，可以将依赖于传统系统的大型机应用程序（例如 IBM 客户信息控制系统 (IBM CICS) 和信息管理系统 (IMS)）重新托管到 Azure 上。

Micro Focus 与 Microsoft 合作退出了令人瞩目的大型机迁移和重新托管工具。 你可以在 Azure 和 Microsoft Windows 平台上，跨开发、测试和生产方案运行大型机应用程序。

Micro Focus 提供开发重新托管和测试重新托管解决方案，可将现有的 COBOL 和 PL/I 业务应用程序扩展到 Azure。
