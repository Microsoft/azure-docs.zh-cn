单击“文件”->“生成设置”。 依次单击“通用 Windows 平台”、“切换平台”。 如果 Unity 编辑器指出需要先下载某些组件，请下载并安装这些组件。 按照下面的屏幕截图配置生成设置。

:::image type="content" source="./media/object-anchors-quickstarts-unity/aoa-unity-build-settings.png" alt-text="生成设置":::
