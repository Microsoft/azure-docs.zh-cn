## <a name="build-and-run-the-app"></a>生成并运行应用

### <a name="build-the-sample-scene"></a>生成示例场景

在 Unity 编辑器中，导航到“Assets/MixedReality.AzureObjectAnchors/Scenes”，然后打开“AOASampleScene”，并将其添加到“场景生成”列表中。
