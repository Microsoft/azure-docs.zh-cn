验证是否可以从 Azure 门户仪表板中查看虚拟机的有关数据。

1. 在 Azure 门户中，选择“仪表板”。

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="Azure 门户导航到仪表板":::

1. 在仪表板页面上，选择“简单的 VM 仪表板”。

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="导航到“简单的 VM 仪表板”":::

1. 查看 Azure 资源管理器模板（ARM 模板）创建的仪表板。 你可以看到某些内容是静态的，但也有一些图表显示了你在一开始创建的 VM 的性能。

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="查看简单的 VM 仪表板":::
