## <a name="set-up-your-device"></a>设置设备

若要将应用部署到 HoloLens，需要将 HoloLens 与计算机配对。

1. 在 HoloLens 中，导航到“设置”->“更新和安全”->“面向开发人员”。
2. 单击“配对”，将屏幕保持打开状态，直到在第一次部署期间将 PIN 输入到 Visual Studio 中。
