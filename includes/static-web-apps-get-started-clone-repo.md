## <a name="clone-the-repository"></a>克隆存储库

在你的 GitHub 帐户中创建存储库后，使用以下命令将项目克隆到本地计算机。

```bash
git clone https://github.com/<YOUR_GITHUB_ACCOUNT_NAME>/my-first-static-web-app.git
```

确保将 `<YOUR_GITHUB_ACCOUNT_NAME>` 替换为你的 GitHub 用户名。
