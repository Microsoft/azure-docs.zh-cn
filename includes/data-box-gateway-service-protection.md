- 若要访问 Azure Stack Edge 服务，组织需要具有企业协议 (EA) 或云解决方案提供商 (CSP) 订阅。 有关详细信息，请参阅[注册 Azure 订阅](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/)。
- 由于此管理服务是在 Azure 中托管的，因此受 Azure 安全功能的保护。 有关 Azure 提供的安全功能的详细信息，请转到 [Microsoft Azure 信任中心](https://azure.microsoft.com/support/trust-center/security/)。
- 对于 SDK 管理操作，你可以在“设备属性”中获取资源的加密密钥。 只有拥有 Resource Graph API 的权限，才能查看加密密钥。
