> 如果你的管理员已为你的组织启用了新的“我的应用”预览体验，那么当你选择帐户图标以查看“我的应用”页面顶部的帐户信息时，会看到一个链接 。 如果选择“尝试‘我的应用’的全新体验!”，将立即转到新的 URL (https://myapplications.microsoft.com) 和已更新的体验。 此后，每次返回到“我的应用”页面时，你将自动进入新体验。<br><br>若要停止使用新体验，可以在“我的应用”页面顶部选择帐户图标以查看帐户信息时，选择“退出预览体验” 。 你会立即返回到旧 URL (myapps.microsoft.com) 和旧体验。 此后，每次返回到“My Apps”页面时，你将自动进入旧体验。
