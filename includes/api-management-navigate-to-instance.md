## <a name="go-to-your-api-management-instance"></a>转到你的 API 管理实例

1. 在 Azure 门户中搜索并选择“API 管理服务”  。

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-1.png" alt-text="选择“API 管理服务”":::

1. 在“API 管理”服务页上，选择你的 API 管理实例。

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-2.png" alt-text="选择你的 API 管理实例":::
