### <a name="running-multiple-containers-on-the-same-host"></a>在同一主机上运行多个容器

如果你想要使用公开的端口运行多个容器，请确保使用不同的公开端口运行每个容器。 例如，在端口 5000 上运行第一个容器，在端口 5001 上运行第二个容器。

您可以让此容器，同时在主机上运行不同的认知服务容器或可以有多个容器的同一个运行的认知服务容器。
