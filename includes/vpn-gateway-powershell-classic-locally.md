使用经典部署模型时，会无法使用 Azure Cloud Shell。 相反，用户必须在计算机上本地安装最新版本的  Service Management (SM) PowerShell cmdlet。 这些 cmdlet 不同于 AzureRM 或 Az cmdlet。 要安装 SM cmdlet，请参阅“[安装服务管理 cmdlet](/powershell/azure/servicemanagement/install-azure-ps)”。 有关一般 Azure PowerShell 的详细信息，请参阅 [Azure PowerShell 文档](/powershell/azure/)。
