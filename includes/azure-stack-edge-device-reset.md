若要使用本地 Web UI 重置设备，请执行以下步骤。

1. 在本地 Web UI 中，转到“维护”>“设备重置”。
2. 选择“重置设备”。

    ![重置设备](media/azure-stack-edge-device-reset/device-reset-1.png)

3. 出现确认提示时，查看警告，并选择“是”以继续。

    ![确认重置](media/azure-stack-edge-device-reset/device-reset-2.png)  

重置会清除设备数据磁盘中的数据。 此过程大约需要 30-40 分钟，具体取决于设备上的数据量。
